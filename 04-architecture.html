State Management

We implemented a centralized state object that tracks GPA, Social Energy, and Academics. Every choice triggers a weight-based recalculation, updating the UI instantly without page reloads.

The "Time Loop" Controller

The simulator uses a progress-based timeline. As tasks are completed, the global "TimeProgress" variable increments, triggering grade transitions (e.g., from Sophomore to Junior).

Interactive Map Logic

The map isn't just a static image—it's a coordinate-based navigation system. Each click triggers a 'Travel' event that consumes Energy while unlocking location-specific tasks.

AI Advisor Logic

The AI Chatbot uses a preset context-aware database to provide "Graduate Advice." It analyzes the user's current GPA and Energy to provide tailored planning tips.

// Task Resolution Logic Example
function resolveTask(taskId) {
  const task = currentTasks.find(t => t.id === taskId);
  state.GPA += task.impact.academics * 0.05;
  state.Energy -= task.impact.effort;
  // Update UI Rings
  renderStats(state);
  checkAdmissionStatus();
}
