Overview

XJTLU Journey Simulator is an interactive game that simulates the postgraduate application process for XJTLU students. It aims to help students understand the various challenges and decisions involved in the application journey.

Core User Needs

  • Understand various choices and consequences in the application process
  • Balance academics, social activities, and mental health
  • Learn time management and resource allocation
  • Experience different postgraduate application paths for various majors
  • Understand how to improve application competitiveness

Game Features

  • Realistic campus map and buildings
  • Diverse events and choices
  • Dynamic status management system
  • Rich ending system
  • Visual novel style story presentation
